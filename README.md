# sample-rails-slack-api とは

Ruby on Rails でSlack APIを利用する際のサンプルです。

## 各バージョン
| 名前 | バージョン |
| ---- | ---- |
| Ruby  | 3.1.3p185 |
| Rails | Rails 7.0.4 |
| MySQL | 8.0.31 | 

## 最初にやること

1. リポジトリのクローン   
`git clone git@github.com:nna1016/sample-docker-rails7.git`
2. Docker 起動
`docker-compose up --build`
